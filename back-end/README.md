## Projeto backend
API Rest para ser utilizada com a aplicação Vue JS

## Configurações

``` bash
# instalar dependencias
composer install
```

### Executar as migrations
``` bash
php artisan migrate
```

### Executar os seeders
``` bash
php artisan db:seed
Email: admin@piperun.com.br
Senha: admin
```

### Executar o servidor
``` bash
php artisan serve
```
